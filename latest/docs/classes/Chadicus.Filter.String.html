Split a string by the specified delimiter.

explode(string $value, string $delimiter = ' ', integer $limit = PHP_INT_MAX) : array
static

@see \explode() This method exists as a filter because filters require the string to be the first argument in the method call.

throws

Thrown if $value is not a string.

Thrown if $delimiter is not a string or is empty.

Thrown if $limit is not an integer.

Arguments

$value

string

The input string.

$delimiter

string

The boundary string.

$limit

integer

If set and positive, the returned array will contain a maximum of limit elements with

                      the last element containing the rest of string.
                      If set and negative, all components except the last -limit are returned.
                      If zero, then this is treated as 1.

Response

array
